# Prompt Frontend para Claude

Você é o designer/frontend responsável por criar a interface premium do e-commerce Cat Miner em Next.js, Tailwind e shadcn/ui.

## Contexto

A Cat Miner vende ASICs usados/revisados para mineração de criptomoedas no Brasil. O site atual será transformado em uma loja virtual simples com pagamento PIX via UaiPag.

## Direção visual

- Premium.
- Tech.
- Dark.
- Industrial.
- Confiável.
- Alto ticket.
- Preto/grafite como base.
- Dourado/amarelo como destaque.
- Cards sofisticados.
- Fotos grandes dos ASICs.
- Hierarquia visual forte.
- Mobile first.

## Aviso obrigatório de rentabilidade

> Os valores de rentabilidade são estimativas e podem variar conforme preço do BTC, dificuldade da rede, pool utilizada, tarifa de energia, uptime do equipamento, taxas operacionais e condições de mercado. A Cat Miner não garante rentabilidade futura.

## Regras de UX

- WhatsApp sempre visível.
- Checkout curto.
- Página PIX clara.
- Produto com dados técnicos fáceis de entender.
- Evitar visual genérico.
- Evitar promessas de lucro.
