# Contexto Consolidado do Projeto

## Nome

Cat Miner E-commerce MVP.

## Objetivo

Transformar a vitrine atual da Cat Miner em um e-commerce simples de ASICs com pagamento PIX via UaiPag.

## Times/ferramentas

- Codex: backend, banco, pagamentos, APIs, segurança, testes.
- Claude: frontend, UI, UX, copy e refinamento visual.

## Produto de alto ticket

A loja precisa passar confiança. Não pode parecer improvisada.

## Compra consultiva

Mesmo com e-commerce, WhatsApp deve ficar visível em todos os pontos críticos.

## Pagamento

Pix-first via UaiPag.

## Risco jurídico/comercial

Não prometer rentabilidade. Sempre usar “estimativa” e “simulação”.

## Estoque

Baixar estoque apenas após pagamento confirmado.

## Admin

Precisa permitir operação básica:

- cadastrar produto;
- editar preço/estoque;
- ver pedidos;
- alterar status de envio.

## V2 futura

- Frete automático.
- NF.
- Área do cliente.
- Cupons.
- Simulador de mineração.
- CRM/WhatsApp automático.
