# Design System — Cat Miner MVP

## Estilo

- Premium.
- Tech.
- Dark.
- Industrial.
- Alto contraste.
- Pouca poluição visual.
- Componentes sólidos e confiáveis.

## Paleta sugerida

```txt
Background principal: #070A0F
Background secundário: #0D1117
Card: #111827
Border: #1F2937
Texto principal: #F9FAFB
Texto secundário: #9CA3AF
Destaque dourado: #F5B301
Destaque âmbar: #F59E0B
Sucesso: #22C55E
Erro: #EF4444
```

## Tipografia

Sugestões:

- Inter.
- Sora.
- Space Grotesk.

## Componentes visuais

### ProductCard

- Imagem grande.
- Nome.
- Hashrate.
- Consumo.
- Preço.
- Selo usado/revisado.
- Selo garantia.
- Botão comprar.
- Botão WhatsApp menor.

### TechnicalSpecs

Grid com:

- Hashrate.
- Consumo.
- Tensão.
- Marca.
- Modelo.
- Garantia.
- Condição.

### ProfitabilitySimulationBox

Box com destaque, mas com disclaimer visível.

### PixPaymentBox

- QR Code central.
- Código copia e cola.
- Botão copiar.
- Status.
- Contador de expiração.

## Botões

Primário:

- Comprar com Pix.
- Dourado/âmbar.

Secundário:

- Falar com especialista.
- Outline.

Danger:

- Cancelar/arquivar admin.

## Layout

Max width:

```txt
1200px ou 1280px
```

Espaçamento:

```txt
Seções: 64px a 96px desktop
Seções: 40px a 56px mobile
```

## Mobile

- CTA fixo ou bem visível na página de produto.
- Carrinho fácil de acessar.
- Checkout com uma coluna.
- Campos grandes.
