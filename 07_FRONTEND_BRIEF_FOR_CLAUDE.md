# Briefing Frontend para Claude

Este arquivo é a versão resumida para enviar ao Claude implementar o front.

## Projeto

E-commerce Cat Miner para venda de ASICs usados/revisados no Brasil, com pagamento PIX via UaiPag.

## Direção visual

- Premium.
- Tech.
- Dark.
- Industrial.
- Confiável.
- Alto ticket.
- Preto/grafite como base.
- Dourado/amarelo como destaque.
- Cards com bordas finas.
- Ícones técnicos.
- Fotos grandes dos ASICs.
- Layout responsivo e limpo.

## Aviso de rentabilidade

Usar em páginas de produto e checkout:

> Os valores de rentabilidade são estimativas e podem variar conforme preço do BTC, dificuldade da rede, pool utilizada, tarifa de energia, uptime do equipamento, taxas operacionais e condições de mercado. A Cat Miner não garante rentabilidade futura.

## UX do checkout

O checkout deve ser simples:

1. Dados pessoais.
2. Entrega.
3. Revisão do pedido.
4. Pagamento via Pix.

Não esconder WhatsApp. Produto de alto ticket precisa de suporte humano visível.

## Tela PIX

A tela de pagamento deve mostrar:

- Número do pedido.
- Valor total.
- QR Code.
- Pix Copia e Cola.
- Botão copiar.
- Status do pagamento.
- Botão verificar pagamento.
- WhatsApp para suporte.

## Mobile

Priorizar mobile first. Botões devem ser grandes, com boa hierarquia, especialmente no produto, carrinho e checkout.
