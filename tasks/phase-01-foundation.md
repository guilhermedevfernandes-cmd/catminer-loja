# Fase 1 — Fundação

## Objetivo

Criar a base técnica do projeto.

## Tarefas

- [ ] Criar projeto Next.js com TypeScript, App Router, Tailwind e src-dir.
- [ ] Instalar dependências principais.
- [ ] Inicializar shadcn/ui.
- [ ] Criar estrutura de pastas.
- [ ] Configurar Prisma.
- [ ] Criar `src/lib/db.ts`.
- [ ] Criar `.env.example`.
- [ ] Criar README inicial do projeto.
- [ ] Configurar scripts no package.json.
- [ ] Garantir que `npm run build` funciona.

## Entregáveis

- Projeto rodando localmente.
- Estrutura de pastas pronta.
- Prisma inicializado.
- shadcn/ui funcionando.

## Comandos úteis

```bash
npm run dev
npm run build
npx prisma init
npx prisma generate
```
